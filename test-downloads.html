YAPE Extension Test Page

How to Test the Notification Feature:

  1. Make sure your YAPE extension is loaded and PyLoad is running
  2. Right-click on any download link below
  3. Select "Send to PyLoad" from the context menu
  4. Check that the notification shows the actual filename from PyLoad (not just "Download link" or button text)
  5. Open the YAPE popup to verify the download appears with the correct name

Expected Behavior: The notification should display the real filename that PyLoad extracted from the URL, not the link text or a generated package name.

Verification Checklist

For each test case, verify:

  • ✓ The notification appears after right-clicking and selecting "Send to PyLoad"
  • ✓ The notification title shows "Download Added"
  • ✓ The notification message shows the actual filename extracted by PyLoad
  • ✓ The notification does NOT show generic text like "Download link" or button text
  • ✓ The download appears in the YAPE popup with the correct filename
  • ✓ Console logs show the debug messages about fetching actual filename

Debugging Tips

To see debug logs:

  1. Open Chrome DevTools (F12)
  2. Go to the Console tab
  3. Look for messages starting with [YAPE-DEBUG]
  4. You should see logs about fetching queue data and using actual filename

Alternative: Check Service Worker logs:

  1. Go to chrome://extensions/
  2. Find YAPE extension
  3. Click "service worker" link
  4. Watch the console for [YAPE-DEBUG] messages
